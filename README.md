# 🎯 Crack Your Internship - DSA Challenge

Welcome to the **Crack Your Internship** repository! This project contains comprehensive solutions to a **280+ DSA (Data Structures and Algorithms)** question sheet designed by **Arsh Goyal**, completed over a **45-day challenge**.

## 📋 Overview

This repository is a curated collection of problem solutions covering essential DSA concepts needed to ace technical interviews for internships and entry-level positions. Whether you're preparing for coding interviews at top tech companies or strengthening your problem-solving skills, this resource will help you master the fundamentals.

## 🎓 Course Details

- **Total Problems**: 280+
- **Duration**: 45 days
- **Difficulty Level**: Beginner to Intermediate
- **Topics Covered**: Arrays, Strings, Linked Lists, Stacks, Queues, Trees, Graphs, Dynamic Programming, Sorting, Searching, and more

## 📂 Repository Structure

The repository is organized by days:
```
├── 1st Day/
├── 2nd Day/
├── ...
├── 10th Day/
├── ...
├── 27th Day/
└── README.md
```

Each day's folder contains:
- Problem statements
- Solutions with explanations
- Multiple approaches where applicable
- Time and Space Complexity analysis

## 🚀 How to Use This Repository

1. **Clone the Repository**
   ```bash
   git clone https://github.com/SohamMisra/-CrackYourInternship.git
   cd -CrackYourInternship
   ```

2. **Browse by Day**
   - Navigate to the respective day folder (e.g., `1st Day`, `2nd Day`, etc.)
   - Each folder contains problems for that day

3. **Study the Solutions**
   - Read through the problem statement
   - Understand the approach and algorithm
   - Review the code implementation
   - Analyze time and space complexity

4. **Practice**
   - Try solving the problems yourself first
   - Compare with the provided solutions
   - Experiment with different approaches
   - Optimize for better time/space complexity

## 💡 Topics Covered

- **Arrays & Strings**: Basic operations, searching, sorting
- **Linked Lists**: Insertion, deletion, reversal, detection
- **Stacks & Queues**: Implementation, applications

- **Graphs**: BFS, DFS, shortest paths
- **Dynamic Programming**: Optimization problems, memoization
- **Sorting & Searching**: Various algorithms and techniques
- **And Much More!**
